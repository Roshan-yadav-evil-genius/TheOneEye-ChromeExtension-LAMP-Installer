TheOneEye - LAMP Installer

What is LAMP? LAMP stands for LinkedIn Amplifier. It is a Chrome extension developed by the TheOneEye team. The extension scores posts and profiles according to your intent, helping you filter opportunities and high-value people from noise.

Linux / macOS

curl -sSL http://__HOST__/install.sh | bash

Windows (PowerShell as Administrator)

irm http://__HOST__/install.ps1 | iex

If you get an error like "running scripts is disabled on this system", run the following command:

Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned -Force

Installation Tutorial

After you run the install command for your system, the extension is placed on your desktop. Follow the tutorial below to turn on Developer mode in Chrome and load the extension.

Files

install.sh   — Linux / macOS installer
install.ps1  — Windows installer (PowerShell)
